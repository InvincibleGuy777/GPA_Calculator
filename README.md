# GPA_Calculator
本程序支持导入和导出Excel表格数据，但是对于其格式有一定要求：

1. 仅支持扩展名为*.xls或*.xlsx的导入与*.xls的导出；
2. 表格格式：从左到右共三列，字段分别为：课程名(随意)，课程学分，课程成绩，注意顺序！
3. 如果一个excel文件中含有多张表，最好把待导入的表格放在最前面

一次录入，终身使用
